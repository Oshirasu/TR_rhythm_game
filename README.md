# Friday Night Funkin': Akage

AKAGE: Reimagined (Demo) に着想を得て制作している、非公式のファン再現プロジェクトです。  
Web ブラウザ上で動く形で、元 mod の演出や挙動をできるだけ再現することを目標にしています。

## 概要

- ブラウザで動作するリズムゲーム実装です
- HTML / CSS / JavaScript で構成されています
- 元 mod の見た目や演出を参考にしつつ、Web 向けに再実装しています

## 状態

現在も調整中のプロジェクトです。  
演出、座標、アニメーション、UI、譜面再現などは今後も更新される可能性があります。

## 起動方法

このプロジェクトは静的サイトです。  
ローカルで確認する場合は、プロジェクト直下の `index.html` を開くか、簡易サーバーで配信してください。

GitHub Pages への公開手順は [DEPLOY_GITHUB_PAGES.md](DEPLOY_GITHUB_PAGES.md) を参照してください。
