## そのまま読める短い発表原稿

こんにちは。  
今回作った作品は、Friday Night Funkin' の mod を参考にして、ブラウザ上で動くようにしたリズムゲームです。

目標は、ただ似た画面を作ることではなく、実際に遊べる作品として完成させることでした。  
そのために、ノーツ判定、長押しノーツ、体力バー、キャラクター切り替え、演出イベント、リザルト画面まで実装しました。  
達成度は自分では 90% くらいだと思っています。

特に工夫したのは、譜面のイベントに合わせてキャラクターや演出を切り替える処理と、Akage の雰囲気に寄せたリザルト画面です。  
譜面データに入っているイベントを JavaScript で読み取って、指定した時間でキャラクターの変更や移動を行うようにしました。  
また、リザルト画面は HTML で構造を作り、CSS で赤・白・黒の配色や手書きっぽい枠、斜線などを調整して、作品全体の雰囲気に近づけました。

制作の中では、自分で調べて書いた部分も多いですが、難しいバグの修正やデザインの調整では ChatGPT に具体的に症状を伝えて相談しながら進めました。  
また、サポートを受けながら問題を分解して考えることも学べました。

TR+ に参加した理由は、プログラミングをただ学ぶだけでなく、作品として完成させて発表する経験をしたかったからです。  
これからは、コードの整理や、別の曲や譜面への対応などを進めて、もっと拡張できる作品にしていきたいです。
